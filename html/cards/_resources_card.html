Resources

These are some very helpful sites and resources vetted by our members. There is a little of everything in this list.

  • HTML & CSS

    • Code Academy HTML & CSS: Hands on tutorial on basic HTML and CSS.
    • Code School HTML & CSS: Hands on tutorial on basic HTML and CSS.
    • HTML essential training: General tutorial on all the basics of HTML.
    • CSS Tricks: Collection of CSS techniques, and uses. Great resource for discovering features and design ideas.
    • Sass: A CSS compiler that offers features such as variables and mixing for CSS.
    • Bourbon.io: A lightweight Sass library.

  • JavaScript

    • Code Academy JavaScript: Hands on tutorial on basic JavaScript.
    • Code School JavaScript: Hands on tutorial on JavaScript. Basic to advance topics. Includes chapters on jQuery and several JavaScript frameworks.
    • Lynda.com JavaScript: General tutorial on the basic of JavaScript.
    • jQuery: A general purpose JavaScript library.

  • Tools and software

    • Git: A version control tool.
    • Code School Git: Tutorial on basic and advance Git concepts.
    • GitHub: A Git based web-hosting service.
    • Grunt JS: A JavaScript task/build runner.
    • Gulp JS: A Javascript task/build runner

  • Backend

    • Ruby on Rails Book: A free online book for learning Ruby on Rails.
    • Code School Rails: Basic and advance Ruby on Rails concepts.
    • Code Academy Ruby: Hands on tutorial on the basics of Ruby.
    • Code Academy PHP: Hands on tutorial on the basics of PHP.

  • Technical

    • A List Apart: Blog with article on everything from front end design, backend management and client interactions.
    • Trent Walton: Blog on web design.

Quick note for VT students: You should all have unlimited access to Lynda.com. This web page offers tutorials on pretty much everything. As a VT student you can log in with your PID here and have access to all of their tutorials.
